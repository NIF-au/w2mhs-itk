# w2mhs-itk
An implementation of the W2MHS white matter hyperintensity segmentation method in ITK



Install/compile instructions for Ubuntu:

# Install opencv

   $ apt-get install libopencv-dev


Build:

# create a build dir, cmake and make
   
   $ mkdir build
   
   $ cd build
   $ cmake ..
   
   $ make
